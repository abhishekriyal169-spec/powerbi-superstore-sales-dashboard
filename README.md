\# Super Store Sales Dashboard (Power BI)



\## Project Overview

This project is a Power BI dashboard built using a synthetic Super Store sales dataset. It provides insights into regional sales, profit margins, product performance, and monthly trends.



\## Dataset

\- Source: Publicly available Super Store Excel dataset

\- Type: Synthetic / practice data

\- Format: Excel (.xlsx)



\## Key Insights

\- Regional sales distribution across the US

\- Profit margin comparison by product category

\- Monthly sales trend analysis

\- Sales vs profit by goods category

\- Order size contribution by category



\## Dashboard Features

\- Interactive regional map (US)

\- Bar charts for profit margin and category-wise performance

\- Line chart for monthly sales trend

\- Donut chart for order size distribution

\- Product-level filtering



\## Tools Used

\- Power BI

\- Microsoft Excel
